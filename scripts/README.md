# Code Index Tool Scripts

This directory contains utility scripts to assist with code indexing and management tasks.

## Available Scripts

### `setup_venv.sh`
Sets up a Python virtual environment using `uv` for the code index tool.

### `run_search_validation.sh`
Runs search validation tests to verify indexing and search functionality.

## Usage Examples

### Set up development environment
```bash
./scripts/setup_venv.sh
```

### Run search validation
```bash
./scripts/run_search_validation.sh
```

## Notes

- All scripts are standalone and do not require subdirectories
- Scripts assume `uv` is available for virtual environment management
- For more information on the main tool, see the root README.md
